# ParentingApp

A millennial-friendly parenting companion app that guides new parents through their child's development journey.

## Overview

ParentingApp helps parents track milestones, monitor growth patterns, and access expert guidance - all in one intuitive mobile experience.

## Key Features

- **Milestone Tracking** - Track developmental milestones by age
- **Growth Charts** - Monitor height, weight, and health metrics
- **Daily Tips** - Personalized, age-appropriate parenting advice
- **Photo Timeline** - Capture and organize precious memories
- **Activity Tracking** - Sleep, feeding, and diaper logs
- **AI Assistant** - 24/7 parenting Q&A support

## Tech Stack

- **Mobile**: React Native (Expo)
- **Backend**: NestJS + PostgreSQL
- **Infrastructure**: AWS/Docker/Kubernetes
